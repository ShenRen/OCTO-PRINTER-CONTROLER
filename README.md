# OCTO-PRINTER-CONTROLER
3D PRINTER OCTTO PRINTER CONTOLER PCB


1. DC 7~35V INPUT(DC12V or DC24V, 5V 4A OUT
2. PSU CONTROL( EXTERNAL RELAY CONTROL 12V or 24V Relay)
3. RGB LED CONTROL
4. Raspberry SPI TOUCH LCD SUPPORT
